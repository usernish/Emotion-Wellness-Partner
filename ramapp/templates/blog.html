About the Project

This web-based application is developed using the Python Django framework. The main objective of this project is to create a tool that promotes emotional well-being by detecting real-time emotions and providing personalized recommendations.

Features

  • Real-time Emotion Detection using OpenCV
  • Music Recommendations from Spotify
  • Task Recommendations to Enhance Mood

Emotion Detection

The application uses the FER 2013 dataset to detect human emotions in real-time. OpenCV is utilized for processing and analyzing facial expressions to determine the user's emotional state.

Music Recommendations

To address copyright issues, the application integrates with Spotify to provide users with personalized music recommendations based on their detected emotions.

Task Recommendations

The task recommendation mechanism suggests simple tasks tailored to the user's emotional state. For example, if a user is feeling sad, they might be encouraged to "go for a walk" or "talk with loved ones" to improve their mood.

Future Enhancements

Potential future enhancements to the tool could include expanding the music library, integrating additional emotion detection methods, and refining the task recommendation algorithm.
