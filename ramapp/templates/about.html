We Offer You Emotional Wellbeing

This is a web-based application which is created using the Python Django package. The main objectives of this project is to develop an Emotion well-being tool that is fully accessible and usable for users. This app detects human real-time emotion and suggests music and tasks which helps to enhance their emotion. The emotion detection mechanism works using Opencv. Here we use a dataset named FER 2013, including photos of different emotions, to detect emotion. In the case of music recommendation, we are not trying to develop a separate music library because of the music copyright issue, so we are trying to connect our app with Spotify. The task recommendation mechanism is a set of simple tasks that helps the users to enhance their mood. For example, if the user is sad we suggest them a task like "go for a walk", or "talk with your loved ones" etc.
